Login

Oauth 2.0 Google is enabled

You have access to 3 kinds of users Admin, Project Manager and Developer.

Admin: has complete access to manage the app.

Project Manager: has access to everything but to edit or delete users accounts.

Developer: has access to the projects where they have been assign.

What can you do in this app?

As a Admin or Project Manager you have access to all the projects in the database you can Create, Read, Update and Delete. On each project you can add Developers that will get access to the project only if they have been added to the project. You can also create Tickets for the projects and assign Developers to the tickets. On each Ticket you have access to add comments.
